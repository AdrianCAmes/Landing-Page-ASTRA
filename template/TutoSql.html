Crear una base de datos

  1. Crearemos una base de datos que tenga dos tablas y una referencia, en este caso tenemos jugador y equipo
  2. Ahora abriremos el command prompt para poder inicializar las tablas

  5. USE TestData
    GO
  6. CREATE TABLE Equipo ( Liga int NOT NULL, IdEquipo int NOT NULL, IdEstadio int NOT NULL, CONSTRAINT Equipo_pk PRIMARY KEY (IdEquipo) );
  7. CREATE TABLE Jugador ( IdJugador int NOT NULL, Edad int NOT NULL, IdEquipo int NOT NULL, CONSTRAINT Jugador_pk PRIMARY KEY (IdJugador) );

Como meter y extraer datos de una base de datos

  1. INSERT dbo.Equipo (IdEquipo, IdEstadio, Liga)
    VALUES (1, 2, 'El Libertador')
    GO
  2. UPDATE dbo.Equipo
    SET Liga = 'Libertadores'
    WHERE IdEquipo = 1
    GO
  3. SELECT IdEquipo, IdEstadio, Liga
    FROM dbo.Equipo
    GO

Como extraer un backup de una base de datos
Usando Transact-SQL

USE SQLTestDB;
GO
BACKUP DATABASE SQLTestDB
TO DISK = 'c:\tmp\SQLTestDB.bak'
WITH FORMAT,
MEDIANAME = 'SQLServerBackups',
NAME = 'Full Backup of SQLTestDB';
GO

Esto es solo para linux

sqlcmd -S localhost -U SA -Q "BACKUP DATABASE [demodb] TO DISK = N'/var/opt/mssql/data/demodb.bak' WITH NOFORMAT, NOINIT, NAME = 'demodb-full', SKIP, NOREWIND, NOUNLOAD, STATS = 10"
